# Airgun Repair Tracker (Demo)

Premium mobile-first repair tracking system for air rifles and pistols.

## Phase 1 – Demo
This version is designed for:
- Testing UI and workflow
- Mobile-first use (phone / iPad)
- QR code generation
- Booking-in flow
- Make → model memory
- Light / dark mode toggle

All data is stored locally in the browser (demo only).

## Features
- Book in repairs with auto capital letters (email excluded)
- Make and model dropdowns with memory
- Serial number based records
- QR code generation (serial number)
- Repair status workflow
- Premium modern workshop UI
- Mobile-friendly design

## Hosting
This demo is intended to be hosted on GitHub Pages (HTTPS).
